# 🎓 Career Counsellor Chatbot

A smart AI-powered Career Counsellor Chatbot built to help students and professionals explore suitable career paths based on their interests, skills, educational background, and goals.

## 📌 Project Overview

Choosing the right career can be challenging. This chatbot aims to simplify the decision-making process by providing personalized career guidance through an interactive conversation. Users can ask career-related questions, explore different professions, and receive recommendations tailored to their preferences.

## ✨ Features

* 🤖 AI-powered conversational interface
* 🎯 Personalized career recommendations
* 📚 Information about various career options
* 🏫 Guidance on courses, degrees, and certifications
* 💼 Career path exploration and planning
* 🌍 User-friendly and interactive experience
* ⚡ Fast and accurate responses

## 🛠️ Technologies Used

* Python
* Streamlit (for the web interface)
* Large Language Models (LLMs)
* Natural Language Processing (NLP)
* APIs (if applicable)

## 🚀 How It Works

1. The user enters their interests, skills, or career-related questions.
2. The chatbot analyzes the input.
3. AI processes the information and generates suitable career suggestions.
4. The user receives personalized guidance and career insights.

## 🔧 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/career-counsellor-chatbot.git
```

2. Navigate to the project directory:

```bash
cd career-counsellor-chatbot
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run app.py
```
